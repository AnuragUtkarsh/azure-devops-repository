# Azure Administrator (AZ-104) Hands-On Projects Repository

This repository contains a collection of **hands-on Azure projects** built while preparing for the **Microsoft Azure Administrator (AZ-104)** certification.

The goal of this repository is to demonstrate **practical cloud engineering skills** including Azure networking, identity, storage, monitoring, security, and infrastructure automation.

Each project focuses on implementing real-world Azure scenarios using best practices.

---

# Repository Overview

This repository includes **18 practical Azure projects** covering core areas required for the AZ-104 certification.

Topics covered include:

* Virtual Machines
* Azure Networking
* Application Gateway
* Azure Storage
* Azure Monitor
* RBAC and Identity Management
* Azure Key Vault
* Managed Identity
* Private Endpoints
* Secure Architecture
* Infrastructure as Code using Bicep
* Azure Backup and Disaster Recovery

# Secure Architecture Implemented

One of the key projects in this repository implements a **secure Azure architecture** including:

* Application Gateway (Layer 7 Load Balancer)
* Backend Virtual Machine
* Azure Key Vault for secret storage
* Managed Identity authentication
* Private Endpoint for storage
* Virtual Network isolation

This architecture follows **Azure security best practices**.
